CAVE HILL

This week started off the same as before, I am still just doing emails at the minute but I think I am really getting the hang of it now, I know there is still a lot of stuff to learn about them becasue currently I am just copying the previous one and changing the code to fir the new one. That is working right now because they all have the same layouts but once they start to change that wont work.

This week we went for a team building day up Cave Hill. I didn't really expect to be doing something like that at the start of placement.

We started off at the very bottom which isn't something I have done before, I have only ever went from about half way. It was such a good dat for it as well, it was around 26°C so it was roasting. At the start everyone was happy with the weather but as we started to walk up we realised it was making it harder, had to get the sun cream and all out.

Once we got to the top we all had a glass out champagne which was again something I did not expect to be doing while I was in "work". getting back down wasn't as bad and once we got to the bottom we went into belfast castle and had some drinks.

It was a really good day all around, gave me a really good chance to get to know some of the people that I wouldn't usually get a chance to work with.
